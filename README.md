# OrderPilot — landing

Strona-wizytówka (one-page) aplikacji **OrderPilot** — asystenta kuriera (Glovo / Wolt / Bolt Food / Uber Eats).

- **Live apka:** https://play.google.com/store/apps/details?id=com.orderpilot.app
- **Stack:** czysty HTML + CSS + JS (bez frameworków). Hosting: GitHub Pages.
- **Styl:** ciemny „HUD", granat `#0A1220` + pomarańcz `#FF6B2C`.

## Struktura
- `index.html` — treść i sekcje
- `styles.css` — całość stylu
- `script.js` — animacja overlaya w hero, reveal przy scrollu, sticky nav
- `assets/` — logo + prawdziwe zrzuty z Google Play
